# digital-forensics-investigation
Digital forensics lab focused on disk image analysis, evidence recovery, and investigative reporting using Autopsy.

## Overview
This project involved conducting a forensic investigation of a disk image using Autopsy to identify evidence of hacking tools, suspicious activity, and links between the suspect and the examined system.

## Objectives
- Analyze a forensic disk image
- Identify installed hacking tools
- Locate evidence of tool usage
- Examine generated artifacts and files
- Correlate evidence to the suspect
- Produce a professional forensic findings report

## Tools Used
- Autopsy
- Windows XP Disk Image Analysis

## Evidence Identified
The investigation identified multiple hacking-related tools and artifacts including:
- NetCat
- John The Ripper
- Brutus
- Cain & Abel
- Ethereal
- NetStumbler
- CuteFTP

## Skills Demonstrated
- Digital forensics
- Evidence analysis
- Artifact examination
- Log and file analysis
- Investigative reporting
- Chain-of-evidence methodology
- Technical documentation

## Findings Summary
The forensic analysis identified evidence of hacking software, password-cracking utilities, network discovery tools, and files associated with brute-force attack methodologies. System ownership artifacts connected the device to the suspect examined during the investigation.
